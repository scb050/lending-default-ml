## Descripción del Proyecto
Este proyecto desarrolla y evalúa modelos de machine learning para la predicción de incumplimiento crediticio (default).

Se implementaron modelos en:
- Scikit-learn
- PySpark

Se realizó:
- Optimización de hiperparámetros con GridSearch
- Evaluación mediante métricas robustas (Precision, Recall, F1, ROC-AUC)
- Análisis de curvas ROC y Precision-Recall
- Interpretabilidad mediante LIME

## Resultados principales
El modelo Random Forest optimizado alcanzó:
- Accuracy: 0.9204
- F1-score: 0.8070
- ROC AUC: 0.9511

## Conclusión
El modelo demuestra alta capacidad discriminativa y equilibrio entre detección de riesgo y control de falsas alarmas.
